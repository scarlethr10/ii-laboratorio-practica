#1 ¿Qué es un framework?

Un framework es una estructura base que se utiliza como punto de partida para elaborar proyectos con un objetivo específico, esta es una especie de plantilla que nos simplifica la elaboración de una tarea ya que solo debemos complementarlo o eliminar componentes que nos parezcan inecesarios.


#2 ¿Qué es Bootstrap 5 y cuál es su origen?

Bootstrap 5 es un tipo de framework o una biblioteca de herramientas de código abierto para diseños de sitios web o aplicaciones web, el cuál contiende diferentes diseños de tipografias, botones menús de navegación,ect.


Originalmente funcionaba como un marco de trabajo para fomentar la utilización de las herramientas internas. En agosto del 2011, Twitter libero a Bootstrap como una interfaz de código abierto y en febrero del 2012 se convirtió en el proyecto de desarrollo mas popular de Github, actualmente la version más reciente es Bootstrap 5.


#3 ¿Qué es diseño responsive en páginas web?

El responsive en las paginas web significa que se puede hacer visible o accesible dicha página en todos los dispositivos electrónicos: tabletas, computadoras, teléfonos, ect.

El diseño responsive dentro de una pagina web se logra a traves de hojas de CSS y técnicas de programación HTML.


#4 ¿Qué son las Github pages?

Es una forma de albergar nuestros proyectos dentro de una pagina web y se pueden realizar cambios dentro de esta siempre resguardando el historial del proyecto, como su última fecha de modificación.


#5 ¿Cuál es la diferencia entre elemento y componente web?

Los elementos texto, imagenes, audio, animaciones y más son basicos para un sitio web, mientras que los componentes webs son un conjunto de caracteristicas que pueden ser o no añadidos a las especifícaciones HTML de forma que permite su edición dentro del documento o aplicación web.


#6 ¿Cuál es la importancia de documentar o comentar código?

Documentar o comentar un codigo de un programa es determinar suficiente información que explica al ordenador como dbe actuar punto por punto utilizando un lenguaje adecuado para que sepan que hacer y para que los humanos entiendan que esta realizando y por qué.


#7 ¿Por qué existe la sección de atribución en un sitio web?

La sección de atribución existe por que en ella se colocan los links y las referencias de las páginas de las que hemos obtenido imagenes, iconos, textos e incluso la plantilla de un sitio web y esto evita que estemos cometiendo plagio sobre componentes de autores.
