HTML + CSS Lectura y edición de código fuente.


El HTML es el codigo base para crear la estructura de un sitio web, cabecera, contenido, pie de página, barras laterales, etc.Funciona mediante etiquetas predefinidas a las que puedes modificar sus estilos mediante CSS.


Por ejemplo, mediante una regla CSS puedo redefinir el estilo de todas las etiquetas h1 (títulos 1 de mi web) para que a partir de ahora tengan un tamaño de 36 píxeles, color negro y tipografía Open Sans.

Bootstrap 5 Componentes, uso de documentación y plantillas.
